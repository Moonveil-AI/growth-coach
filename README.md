# Growth Coach

A Claude Code skill that turns Claude into a senior growth consultant for technical founders. Describe a business challenge and the skill diagnoses the problem, retrieves battle-tested playbooks via the GrowthPilot API, and writes a personalized action plan.

## Install

Drop [`SKILL.md`](./SKILL.md) into `~/.claude/skills/growth-coach/` and set your API key:

```bash
export GROWTHPILOT_API_KEY=gp_xxxxx
```

Then in Claude Code, run `/growth-coach` and describe your situation.
