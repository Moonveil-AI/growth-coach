# Growth Coach

> AI growth consultant for technical founders. Stops giving generic advice, starts writing plans you can ship this week.

![Growth Coach demo](https://github.com/Moonveil-AI/growth-coach/releases/download/v0.3.0/demo.gif)

A Claude Code skill powered by 200+ battle-tested growth playbooks. Describe a business challenge and Growth Coach diagnoses the problem, picks the right framework, and writes an action plan — with a ready-to-ship asset included (cold email, LinkedIn post, pricing table, whatever fits).

## Install

In Claude Code:

```
/plugin marketplace add Moonveil-AI/growth-coach
/plugin install growth-coach@growth-coach
/growth-coach
```

First run asks for your email and registers a free API key (50 queries, no credit card). After that, just describe your challenge.

## What you get

1. **Diagnosis** — what's actually going on
2. **Strategy** — the right framework for your stage
3. **Action plan** — 3–5 concrete steps, time-boxed
4. **Deliverable** — a ready-to-use asset, not a template

## Coverage

Product-market fit · acquisition (SEO, LinkedIn, cold outreach, PLG, referrals) · pricing · sales · fundraising · indie-hacker workflows · founder wellbeing.
